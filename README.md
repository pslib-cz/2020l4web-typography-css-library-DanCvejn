## Implementation
Just download css file and link it to your html.
```
    <link rel="stylesheet" href="./*/typography.css">
```
"*" = folder where you have css files.
If you have it in the same folder as the html file then use this: 
```
    <link rel="stylesheet" href="./typography.css"> 
```
